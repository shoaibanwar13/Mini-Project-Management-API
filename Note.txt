I Use pipenv to maintain dependecies if you want to run this project locally 
Prepare the virtual environment and install all dependencies using the following commands. 

1) cd ProjectManagementApi

2) pipenv shell

3)pipenv install 

4) python manage.py makemigrations 

5) python manage.py migrate

6) python manage.py runserver
Otherwise you can run this project by install requirements.txt dependecies using this command:

pip install -r requirements.txt
